🧠 REGISTER FLOW (Architecture)
When user hits /register:

Validate input (name, username, email, password)

Check if email already exists

Check if username already exists

Hash password using bcrypt

Save user

Generate JWT

Return token (NOT password)

That’s clean flow.

--------------------------------

🧠 LOGIN FLOW
When user hits /login:

Find user by email

If not found → error

Compare password using bcrypt.compare()

If mismatch → error

Generate JWT

Return token

Simple. Secure. Professional.
